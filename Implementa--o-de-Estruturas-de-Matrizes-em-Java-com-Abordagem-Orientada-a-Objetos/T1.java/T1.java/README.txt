Descrição:
Este trabalho implementa uma abstração para matrizes em Java através da interface Matrix e suas duas implementações: 
- FullMatrix: matriz densa (todos os elementos armazenados explicitamente).
- SparseMatrix: matriz esparsa (apenas elementos não-nulos armazenados em listas circulares encadeadas com nós sentinelas).

A interface Matrix define operações como soma, subtração, multiplicação, acesso por índice, conversão, iteração etc.

---

Atividades concluídas:

✅ A1: Implementação da classe `Vector` com iterador interno.
✅ A2: Finalização das classes `MatrixElement` e `BadDimensionsException`.
✅ A3: Implementação completa das classes `FullMatrix` e `SparseMatrix` conforme especificado.
✅ A4: Criação de testes extensivos na classe `Main`, cobrindo:
  - Todas as operações de ambas as matrizes.
  - Casos esperados de erro com exceções.
